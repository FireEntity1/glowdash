# Glowdash

## Features
    - Highscore upon level complete
    - Procedurally generated obstacles
    - Endless gameplay
    - Colourful shaders
    - Motion blur aesthetics

## Controls
    Press space to jump and click on "play" to start the game! Only a single button is required to play.
